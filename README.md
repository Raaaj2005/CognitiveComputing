# 🧠 Cognitive Computing (UCS420) - Lab Assignments

A comprehensive collection of lab assignments and projects for the **Cognitive Computing (UCS420)** coursework at Thapar Institute of Engineering and Technology (TIET). This repository demonstrates a progressive application of Data Science, Machine Learning, and Natural Language Processing (NLP) techniques using Python.
